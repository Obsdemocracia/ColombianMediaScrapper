# ColombianMediaScrapper


Este proyeco busca crear funciones para descargar noticias de los medios digitales en Colombia


Necesita los paquetes: 

- bs4
- requests
- numpy

Existen funciones para los siguientes medios: 

- El Tiempo
- El Espectador
- RCN Radio
- El Nuevo Siglo
- El País
- W Radio
- Diario Magdalena 
- El Vallenato
- La Nación 
- La Patilla
